# Task 1: Iris Flower Classification

**Objective:
Predict the species of an iris flower based on its sepal and petal measurements.

**Approach:
Performed data exploration and visualization to understand the dataset. Trained a classification model using flower measurements as input features and species as the target variable. Evaluated the model using classification metrics.

**Results and Insights:
The model successfully classified iris flowers with high accuracy. Petal length and petal width were found to be the most important features for distinguishing between species.



# Task 2: House Price Prediction

**Objective:
Predict house prices using features such as area, number of bedrooms, and other property characteristics.

**Approach:
Conducted data preprocessing and exploratory analysis. Applied Linear Regression to model the relationship between house features and price. Evaluated model performance using regression metrics.

**Results and Insights:
The model showed a positive relationship between property size and price. Larger houses with more rooms generally had higher market values.



# Task 3: Customer Segmentation

**Objective:
Group customers into different segments based on purchasing behavior and demographic characteristics.

**Approach:
Performed data cleaning and feature analysis. Applied K-Means Clustering to identify customer groups with similar characteristics. Visualized the clusters for better interpretation.

**Results and Insights:
Distinct customer segments were identified, enabling targeted marketing strategies. Some groups showed higher spending behavior, while others represented budget-conscious customers.



# Task 4: Predicting Insurance Claim Amounts

**Objective:
Estimate medical insurance charges using personal attributes such as age, BMI, smoking status, and number of dependents.

**Approach:
Performed exploratory data analysis and visualized the relationship between key features and insurance charges. Encoded categorical variables and trained a Linear Regression model. Evaluated performance using MAE and RMSE.

**Results and Insights:
Smoking status had the strongest impact on insurance charges, followed by age and BMI. The model provided reasonable predictions and highlighted the factors contributing most to higher insurance costs.



# Task 5: Loan Approval Prediction

**Objective:
Predict whether a loan application will be approved based on applicant information such as income, CIBIL score, assets, and employment status.

**Approach:
Conducted data exploration and preprocessing, including encoding categorical variables. Trained Logistic Regression and Decision Tree classification models to predict loan approval status. Evaluated performance using accuracy and classification metrics.

**Results and Insights:
Applicants with higher CIBIL scores, greater annual income, and stronger asset values were more likely to receive loan approval. The models effectively identified key factors influencing lending decisions and supported data-driven decision-making.
